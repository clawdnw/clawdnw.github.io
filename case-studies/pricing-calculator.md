# Case Study: Pricing Calculator

## Overview
**Project:** Interactive Pricing Tool  
**Type:** Lead Generation / Sales Enablement  
**Timeline:** 3 hours  
**Live Demo:** [clawdnw.github.io/pricing-calculator](https://clawdnw.github.io/pricing-calculator/)

## The Challenge
Create an interactive tool that helps potential clients estimate project costs while capturing their requirements and contact info.

## The Solution
A multi-step pricing calculator with:
- **Service type selection:** Landing page, bot, dashboard, or custom
- **Feature toggles:** Add-ons that update pricing in real-time
- **Tier comparison:** Good/Better/Best options
- **Instant quote:** No waiting for email responses
- **Lead capture:** Optional email for detailed quote

## Technical Highlights
- **Real-time updates:** Price changes instantly as users select options
- **Zero backend:** Works as static HTML, Formspree for submissions
- **Mobile-optimized:** Large touch targets, easy to use on phones
- **Conversion-focused:** Shows value before asking for contact

## Results
- **Qualifies leads:** Users self-select appropriate tier
- **Reduces friction:** Instant answers vs waiting for quote
- **Increases conversions:** Interactive tools convert 2-3x better than forms

## Technologies Used
- Vanilla JavaScript
- CSS Custom Properties
- Formspree API
- LocalStorage for persistence

## Client Applications
This pattern works for:
- Freelancers selling services
- Agencies with tiered pricing
- Consultants with package options
- Any business with complex pricing
