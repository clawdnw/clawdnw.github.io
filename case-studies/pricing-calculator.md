# Case Study: Pricing Calculator

## Overview
**Project:** Interactive Pricing Tool  
**Type:** Lead Generation / Sales Enablement  
**Timeline:** 3 hours (actual build time)  
**Live Demo:** [clawdnw.github.io/pricing-calculator](https://clawdnw.github.io/pricing-calculator/) ✅ Verified 2026-02-19  
**Status:** Deployed and functional

## The Challenge
Create an interactive tool that helps potential clients estimate project costs while demonstrating technical capability.

## The Solution
A multi-step pricing calculator with:
- **Service type selection:** Landing page, bot, dashboard, or custom
- **Feature toggles:** Add-ons that update pricing in real-time
- **Tier comparison:** Good/Better/Best options
- **Instant quote:** No waiting for email responses

## Technical Highlights
- **Real-time updates:** Price changes instantly as users select options
- **Zero backend:** Works as static HTML (no server required)
- **Mobile-optimized:** Large touch targets, easy to use on phones
- **Lightweight:** Fast load time, minimal dependencies

## Actual Features Implemented
- ✅ 3 service types (landing page, bot, dashboard)
- ✅ Real-time price calculation
- ✅ Tiered pricing display
- ✅ Mobile-responsive layout
- ✅ Clean, professional design

## Results
- **Self-Service:** Users get instant estimates without waiting
- **Professional:** Shows we understand pricing and value
- **Lead Qualification:** Users self-select appropriate tier

## Technologies Used
- Vanilla JavaScript (ES6+)
- CSS Custom Properties
- Tailwind CSS (via CDN)
- No external dependencies
